Adomin8ers_MonsterPartyPack

:: REQUIREMENTS ::

:: RECOMMENDED ::
• Adomin8er's HD Vanilla Portraits


:: FEATURES ::
• Custom characters for Vanilla Stardew Valley

:: USE ::
• Add this mod to the Stardew Valley/Mods folder on your system.
• This contains HD portraits for the additional characters. I recommend using my HD Portraits mod to match the aestetic across mods.


:: GENERAL ::
• Images generated via Stable Diffusion and Divine Elegance Mix.
• At the moment only the Dhampir has been added.
